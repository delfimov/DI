# DI Container

Lightweight and nimble PSR-11 PHP Dependency Injection Container

## Requirements
 
 * [PHP >= 5.6](http://www.php.net/) (though, I highly recommend to use PHP 7) 
 * [Composer](https://getcomposer.org/download/)

## Installation

 * Run `composer require delfimov/di`
